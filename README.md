# CASTLE — Cybersecurity And Safety Trace & Lifecycle Engine

Deployment starter. This page just proves the pipeline works before we put
the real app behind it.

## Deploy it (free, ~5 minutes, no credit card)

1. **GitHub**: create a repo (e.g. `castle` or `castle-safety`), upload
   `index.html` and this `README.md` via "uploading an existing file."
2. **Vercel**: sign in at vercel.com with GitHub (free Hobby plan) →
   **Add New → Project** → import the repo → **Deploy**.
3. You'll get a live URL like `https://castle-<something>.vercel.app`
   within a minute.

## Domain ideas to check availability for (at any registrar — Namecheap,
Google Domains, etc. — typically $10–15/year)

Try these, roughly in order of how well they read:
- `castlesafety.com`
- `usecastle.io`
- `getcastle.app`
- `castle-fusa.com`
- `trycastle.dev`

I can't check live availability myself (no registrar access from here) —
these are just strong candidates worth checking first. `.io` and `.app`
domains tend to be more available than `.com` for short brandable names,
but `.com` still reads as more "real product" to enterprise buyers if
that matters for how CASTLE will be positioned.
